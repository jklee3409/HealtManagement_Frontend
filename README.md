# FitWell Frontend

## 프로젝트 소개 📋
FitWell의 프론트엔드 애플리케이션은 React를 기반으로 한 개인화된 건강 관리 웹 서비스입니다.  
로그인, 회원가입, 식단 및 운동 기록, 피드백 분석 등 다양한 UI 기능을 제공합니다.

---

## 기술 스택 🔧
- **언어**: JavaScript (ES6+)
- **프레임워크**: React
- **상태 관리**: Redux
- **스타일링**: TailwindCSS
- **빌드 도구**: Vite
- **API 통신**: Axios

---
